Lab 2: HTML & CSS

This lab is aimed at ensuring that students understand the makeup of the visual aspects of a web page by focusing on basic HTML and CSS. The students will be required to build a simple webpage using HTML and CSS. To highlight the importance of using semantic markup (the appropriate HTML tags), the students are required to only use semantic markup. The students are also encouraged to use CSS Flexbox.

    The div element has no special meaning at all…Authors are strongly encouraged to view the div element as an element of last resort, for when no other element is suitable. Use of more appropriate elements instead of the div element leads to better accessibility for readers and easier maintainability for authors.
